# F1 Race Strategy Analysis (Motorsport Analytics Project)

## Overview
This project analyzes Formula 1 race data to uncover insights into race strategy, tire performance, and lap time trends.  
The goal is to simulate how data-driven decision making can improve race outcomes in motorsport.

---

## Objectives
- Analyze lap time degradation across tire compounds (Soft, Medium, Hard)
- Compare stint performance and strategy effectiveness
- Identify optimal race strategies based on data
- Build interactive dashboards for visualization

---

## Tools & Technologies
- Python (Pandas, FastF1)
- Tableau (Data Visualization)
- GitHub (Project Documentation)

---

## Dataset
- Source: FastF1 API
- Race: 2023 Bahrain GP

---

## Key Analysis

### 1. Lap Time Trend
- Visualized lap-by-lap performance
- Identified tire degradation patterns
- Compared drivers across stints

### 2. Tire Compound Performance
- Soft tires: Faster but degrade quickly  
- Medium tires: Balanced performance  
- Hard tires: Slower but more consistent  

### 3. Stint Analysis
- Compared average lap times by stint
- Evaluated pit stop timing impact
- Identified optimal stint length

### 4. Strategy Insights
- Undercut opportunities observed  
- Trade-off between pace vs durability  
- Strategy depends heavily on track + tire degradation

## Key Insights
- Tire degradation is the most critical factor in race performance  
- Early pit stops (undercut) can significantly improve track position  
- Medium compound often provides the best balance for strategy
